This course helps better understanding latest web development technologies including HTML5 and CSS3. Desing and mobile representation will be focused on with Media Queries. Using Javascript frameworks such as jQuery to add behaviors to website will be introduced. With labs and exercises, student will gain hands on experience of web developing. By the end of this course successful students will be able to build dynamic and semantically marked-up content that is viewable on mobile and desktop screen sizes.
